# Bonafide Certificate Generator

A simple static certificate generator for Zilla Parishad High School. It uses one HTML file, Tailwind CDN, plain JavaScript, and `html2pdf.js` for PDF export.

## Use

Open `index.html` in a browser, fill the form, and click **Generate PDF**. The PDF is saved as:

```text
AdmissionNO-student-name.pdf
```

The certificate page is fixed at `22cm x 16cm` for both preview and PDF output.

## Deploy

Upload these files to any static hosting service:

```text
index.html
assets/telangana-logo.png
```

No backend, database, login, or build step is required.
